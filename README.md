# My personal portfolio

Personal portfolio website built with Yii2 PHP framework, Bootstrap 5, and jQuery. Runs on XAMPP locally.
[Already running in joelfaura.com](https://joelfaura.com/)

## Features

- Single-page layout with animated section navigation
- Light/dark theme toggle (persisted via cookie)
- Multilingual support: English, Spanish, Catalan (persisted via session + cookie)
- Contact form with SMTP email delivery
- CV download (language-aware PDF)
- Responsive design with mobile bottom navigation bar
- Hover tooltips on all navigation controls

## Stack

- **Backend:** PHP 7.4+, Yii2 2.0.55
- **Frontend:** Bootstrap 5, jQuery, Font Awesome 5
- **Mail:** SwiftMailer (SMTP)
- **Server:** Apache via XAMPP, MySQL
